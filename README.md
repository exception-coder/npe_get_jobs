## 📖 使用说明

### 第一步：配置公共信息 🔧

在开始使用之前，你需要先完成公共配置，这些配置会应用到所有投递平台：

1. 进入系统后，找到**公共配置**区域
2. 配置以下关键信息：
   - **企业类型筛选**：选择你感兴趣的企业类型（如互联网、国企、外企等）
   - **黑名单设置**：添加你不想投递的公司或关键词
   - **简历信息**：确保你的简历信息完整准确
   - **AI话术偏好**：设置打招呼话术的风格（专业、活泼、简洁等）

![公共配置示例](docs/images/readme/common-config.png)

💡 **小提示**：公共配置只需设置一次，后续可随时调整！

---

### 第二步：配置投递平台 🎯

完成公共配置后，就可以针对具体的投递平台进行个性化配置了：

1. **选择投递平台**
   - 在左侧导航栏选择你想使用的平台（如 BOSS直聘、猎聘、51Job 等）

2. **设置检索条件**
   - **搜索关键字**：输入职位关键词（如"Java开发"、"产品经理"），可用逗号分隔多个关键词
   - **工作城市**：选择你的目标城市（支持多选）
   - **工作经验**：选择符合你的经验范围
   - **职位类型**：全职/兼职/实习
   - **薪资范围**：设置你的期望薪资区间
   - **学历要求**：选择学历要求
   - **行业筛选**：最多选择 3 个感兴趣的行业
   - **公司规模**：选择偏好的公司规模
   - **融资阶段**：选择公司融资阶段偏好

3. **保存配置**
   - 配置完成后，点击**保存配置**按钮

4. **开始投递** 🚀
   - 点击**一键投递**按钮，系统将自动开始为你筛选和投递简历
   - 你可以随时点击**手动加载任务状态**查看投递进度

![平台配置示例](docs/images/readme/platform-config.png)
